## The system

Read README.md fully before touching anything. IM OLD GREG is a
domain-agnostic shape-matching system on a Riemannian manifold.
29 locked relationship types. Wojaks (seed + accretion) are ground
truth; pepos and memos are descriptive only; axes are first-class
and belong to no memo. Storage is the append-only .greg format
(CBOR, witness mesh, flat rhizome). Querying is diffusion, never
traversal.

## Hard rules

- Core logic in Curry. Compiled by KiCS2. No exceptions.
- Geometry through Haskell libraries (hmatrix, algebraic-graphs,
  manifolds). Never hand-roll differential geometry, tensor ops,
  or linear algebra.
- Harness in POSIX shell.
- No Python. No JSON. CBOR everywhere.
- The 29 relationship types are locked. `greg types` must always
  print exactly 29.
- Axes are never scoped to a memo. Memos and pepos never own,
  contain, or constrain wojaks.
- Coordinates are typed and recursive. Never flatten to scalars.
  Never cap dimensions. Field minimum is 31 axes.
- No indexes, schemas, or hierarchy in storage. The rhizome is flat.
- Queries propagate signal; they do not walk graphs.
- The operator's machines cannot compile. GitHub Actions builds
  everything; binaries ship via Releases. A broken build outranks
  all other work.
- Attribution: `{"name":"Weft","role":"co-author"}` only. No
  session links, session IDs, or "Generated by" footers anywhere.
- No em-dashes in prose.

## Layout

- `src/curry/` — the core. Cbor, Axes, Memo, Metrics,
  Canonicalize, Analogy, Query, Storage, Witness, Chores,
  AxisSeeds, ImOldGreg (main).
- `src/haskell/` — greg-geom helper: hex/binary transport,
  numeric kernels (CBOR request in, CBOR reply out).
- `harness/` — shell wrappers; run.sh owns the file boundary.
- `tests/` — Curry test mains; `error` on failure.
- `reference/` — prose definitions of the 29 types.
- `templates/` — starter CBOR payloads.

## Transport contract

The greg binary never opens files. It reads the field as
hex-encoded .greg bytes on stdin and emits either hex (mutations)
or a report (reads) on stdout. greg-geom bin2hex/hex2bin sits at
the file boundary. Keep it this way: it is what keeps .greg
byte-exact on every platform and locale.

## Verification

Local: `curry-frontend --flat -i <curry-base>/src -i src/curry
<module>` typechecks without a full KiCS2. CI compiles with real
KiCS2 (currylang/kics2 Docker image) and runs every test. Run
verification before claiming anything works.
